Group30's WeatherAPP

Group30's WeatherAPP is a school project by some students form UIB. The main goal of the project is to demonstrate socket programming in python, with TCP and UDP. The project consists of a weather station that sends weather data to a server that then stores the data in a file and handles requests, for example to this web app that you are viewing right now. You might ask, whose man is this? And to tell you the truth, we don't know. The server can also handle requests from a client with TCP to a program operated through the terminal.
